# كتابي — منصة الكتب الإلكترونية

## 🚀 طريقة الرفع على GitHub Pages

### الخطوة 1: ارفع الملفات الأربعة على GitHub
```
index.html
style.css
app.js
README.md   ← هذا الملف
```

### الخطوة 2: فعّل GitHub Pages
1. اذهب إلى **Settings** في الـ Repository
2. اختر **Pages** من القائمة الجانبية
3. تحت **Source** اختر: **Deploy from a branch**
4. اختر الفرع **main** والمجلد **/ (root)**
5. اضغط **Save**

### الخطوة 3: انتظر دقيقة
سيظهر رابط موقعك في نفس الصفحة مثل:
`https://username.github.io/repo-name/`

## 📌 ملاحظات
- الموقع يتصل بـ Supabase تلقائياً (قاعدة البيانات جاهزة)
- جميع الصفحات مدمجة في ملف `app.js`
- التصميم الكامل في ملف `style.css`
- لا تحتاج تثبيت أي شيء
